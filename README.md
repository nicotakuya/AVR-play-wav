# AVR play wav
## Overview
AVRマイコンでwavファイルを鳴らすプログラム。「いらっしゃいませ」と喋ります。

## Files
converter : wavファイルを変換するツール。

playwav.c : ソース

voice.h : 音声データ

playwav.atsln : Atmel Studio 6.2用プロジェクトファイル。 

playwav_schematics.png : 回路図。

## Parts
・AVRマイコン ATmega168/328P

・圧電スピーカー
